# 《芯片战争》阅读笔记

## 为什么读这本书

芯片是我国被美国「卡脖子」的领域之一，芯片领域的发展对我们国家很重要。
因此，我想了解芯片产业，包括芯片的发展历史、芯片公司、芯片技术等。
这样可以帮助我更好地认识现状，对相关芯片新闻能有更好的判断能力，
同时也可以分析我是否能从事芯片相关行业的一些工作。

盘点一下我想解决的问题：

- 世界上有哪些实力较强的芯片公司？
- 这些芯片公司是如何崛起的？
  - Intel 是如何崛起的？
  - TI 是如何崛起的？
  - 台积电是如何崛起的？
  - 中芯国际是如何崛起的？
- 美国半导体产业是如何发展的？
- 日本半导体产业是如何发展的？
- 韩国半导体产业是如何发展的？
- ASML 是如何研发出 EUV 光刻机的？
- 美国在芯片制造领域为什么要依赖台积电，而不能自给自足？
- 美国在光刻机领域为什么要依赖 ASML，而不能自己制造光刻机？

## 这本书的大纲是什么

全书共8章，共54节，每节平均6页左右。

这8章是：

- 冷战时期的筹码
- 美国世界的电路
- 失去领导能力？ （介绍了日本半导体的崛起）
- 美国复兴
- 集成电路，集成世界？ （介绍了台积电和中芯国际的崛起）
- 离岸创新 （介绍无晶圆厂和阿斯麦的崛起）
- 中国的挑战 （介绍中国半导体的发展）
- 芯片瓶颈

从目录来看，主要涉及的国家有：

- 美国
- 苏联
- 日本
- 中国（包括中国大陆和中国台湾）

## 这本书的主要内容是什么

### 万维钢的推荐序

- 万维钢提到的「我们的三个错误认知」及作者认为正确的认知
  - 任何高科技都可以用「堆积」的方法获得 => 高端芯片是不可堆积的，要求高水平人才的奇思妙想、整个生态系统和可遇而不可求的机遇
  - 创新应该由政府来主导 => 创新从来都是各个公司像生物演化一样自行冒险探索优胜劣汰出来的
  - 我们应该独立自主 => 让自己变得更值得依赖

- 万维钢提出的「本手」
  - 从低端芯片做起，慢慢往上攀爬
  - 按照发达国家的标准，建立一个真正适合创新的环境，培养真正的创新型人才
  - 进一步改革开放，让全世界像现在都依赖美国一样越来越依赖中国

### 第一部分 冷战时期的筹码

- 第1节 从钢铁到硅片：战争加速了对计算能力的追求，因为更高的精度需要更多的计算

- 第2节 开关：1947年，贝尔实验室的肖克利、布喇顿和巴丁发明了晶体管

- 第3节 诺伊斯、基尔比和集成电路：1958年，TI 的基尔比发明集成电路；1959年，仙童的诺伊斯也独立发明集成电路

- 第4节 起飞：美国的阿波罗11号飞船使用了仙童芯片，美国的「民兵」导弹使用了TI芯片

- 第5节 迫击炮和规模生产：莱思罗普发明光刻技术，张忠谋改进TI的制造工艺，而安迪·格鲁夫则改进仙童的制造工艺

- 第6节 「我......要......发......财」：1965年，摩尔提出摩尔定律，芯片成本下降扩大了其民用市场，没有股权导致仙童员工离职

### 第二部分 美国世界的电路

- 第7节 苏联硅谷：苏联利用情报集团学习和窃取美国半导体技术，并修建了一座半导体城——泽列诺格勒

- 第8节 「复制」策略：苏联采取「复制」美国的芯片设计的策略，但这样其实使他们逐渐落后于美国

- 第9节 晶体管推销员：索尼发明晶体管收音机，夏普发明半导体计算器，日本消费电子产业迅速发展

- 第10节 「晶体管女孩」：仙童在香港、新加坡和马来西亚等亚洲地区进行海外组装，通过廉价劳动力降低组装成本

- 第11节 精准打击：越南战争于美国而言是一个成功的武器试验场，TI的「宝石路」激光制导炸弹准确命中越南的清化大桥

- 第12节 供应链策略：TI在台湾和新加坡等地建造半导体组装工厂

- 第13节 英特尔的革命：1968年，诺伊斯和摩尔成立英特尔，两年后推出 DRAM 芯片

- 第14节 五角大楼的抵消战略：美国国防部大力投资芯片产业，利用精确的制导导弹来「抵消」苏联的数量优势

### 第三部分 失去领导能力

- 第15节 「竞争很激烈」：索尼、东芝、NEC等日本公司的芯片质量超越美国，同时在消费电子产品领域也逐渐夺取市场份额

- 第16节 「与日本的战争」：美国指控日本公司进行间谍活动；日本发展半导体的策略：受保护的国内市场、政府补贴、银行长期贷款

- 第17节 「运送垃圾」：日本的尼康和佳能在光刻机领域打败美国的 GCA 和 Perkin Elmer

- 第18节 20世纪80年代的原油：美国 AMD、Intel、仙童等公司成立美国半导体行业协会，以争取政府的更多支持。

- 第19节 死亡螺旋：美国政府开始支持半导体行业：降低资本利得税、放松金融监管、保护知识产权、强迫日本对DRAM芯片的出口设定配额、成立 Sematech 财团来增加芯片行业的合作

- 第20节 日本可以说「不」：美国在半导体行业存储芯片、光刻机等领域已落后于日本

### 第四部分 美国复兴

- 第21节 芯片之王：辛普劳支持帕金森兄弟创立美光，最终挽救美国的 DRAM 产业

- 第22节 颠覆英特尔：英特尔放弃 DRAM 市场，专注于个人电脑微处理器，格鲁夫采取「精确复制」的新制造方法来改进制造工艺

- 第23节 「敌人的敌人」：韩国的崛起：1938年，李秉喆创立三星，

- 第24节 「这就是未来」：康维和米德首先将芯片设计标准化，雅各布斯和维特比等成立高通，为 DARPA 和 NASA 建造太空通信系统

- 第25节 克格勃T局：美国政府发现苏联的情报集团，加强对先进技术的海关检查

- 第26节 「大规模毁灭性武器」：抵消战略的影响：苏联振兴其芯片制造的努力彻底失败，原因：政治干预、过度依赖军事客户、缺乏国际供应链

- 第27节 战争英雄：「宝石路」激光制导炸弹在海湾战争中发挥巨大作用
  - 这些炸弹有几个晶体管、一个激光传感器和一对绑在旧「笨弹」上的机翼组成

> Question: 几个晶体管能够提供足够的计算能力吗？

- 第28节 「冷战结束了，你们赢了」：日本半导体行业在萎缩，苏联信息产业落后，冷战结束

### 第五部分 集成电路，集成世界

- 第29节 「台湾想要一个半导体产业」：1987年，张忠谋成立台积电，催生无晶圆厂芯片设计公司

- 第30节 「所有人都必须制造半导体」：20世纪七八十年代，中国大陆的芯片产业远远落后于西方

- 第31节 「与中国人分享上帝的爱」：2000年，张汝京创立中芯国际，致力于芯片制造

- 第32节 光刻战争：1996年，英特尔着急六家芯片制造商成立 EUV 联盟，并投入大量资金发展 EUV 光刻技术

- 第33节 创新者的困境：英特尔拒绝为苹果制造芯片，从而错失进军移动设备的机会

- 第34节 跑得更快？：其他国家正在缩小与美国在半导体行业的差距

### 第六部分 离岸创新

- 第35节 「真正的男人要有晶圆厂」：美国芯片行业设计与制造分离的趋势越来越明显，但AMD的创始人桑德斯仍坚持芯片设计与制造一体化
  - 三类芯片：逻辑芯片、存储芯片、模拟芯片

- 第36节 「无晶圆厂的革命」：无晶圆厂的代表：1993年成立的英伟达、1985年成立的高通、Xilinx、Altera

- 第37节 张忠谋的大联盟：AMD 将芯片设计和制造业务分开，格芯诞生。台积电与上下游伙伴构成一个生态

- 第38节 苹果硅：苹果手机的芯片在台积电制造，苹果手机在中国大陆组装

- 第39节 EUV 光刻机：西盟（Cymer）和通快（Trumpf）提供 EUV 光源，蔡司提供透镜，再加上阿斯麦的晶圆定位台，最终得到 EUV 光刻机

- 第40节 「没有B计划」：格芯放弃生产尖端芯片，竞争者剩下台积电、英特尔和三星三家

- 第41节 英特尔如何遗忘创新：英特尔坚持芯片设计与制造结合的集成模式，错过了人工智能芯片设计的基机遇

### 第七部分 中国的挑战

- 第42节 中国制造：中国大陆需要生产自己的芯片

- 第43节 「把冲锋号吹起来」：中国大陆在半导体行业的市场份额只有6%，《中国制造2025》呼吁减少中国队进口芯片的依赖

- 第44节 技术转让：IBM、AMD 和 Arm 向中国转让部分芯片技术，中国在设计和生产数据中心芯片方面对境外的依赖大大降低

- 第45节 「并购势必发生」：紫光在国内大肆并购，收购了展讯通信和锐迪科等公司，但在收购台湾和美国的半导体公司时遇到阻碍

- 第46节 华为的崛起：1987年，华为成立，以交换机起家，雇用 IBM 员工来制定管理流程，进军手机，并设计自己的芯片

- 第47节 5G 未来：5G 将导致更多的设备连接到蜂窝网络，产生更多的数据，从而需要更强大的处理能力

- 第48节 下一个抵消战略：美国试图在微电子技术领域建立新的技术优势，目前看来尚未成功

### 第八部分 芯片瓶颈

- 第49节 「我们正在竞争的一切」：美国发起贸易战，对华为和中兴通讯进行制裁

- 第50节 福建晋华：美光起诉联华电子和晋华侵犯其专利，但影响甚微，于是美国对晋华进行制裁

- 第51节 打击华为：美国制裁华为、中科曙光、飞腾等公司，荷兰不批准向中国公司出售阿斯麦的 EUV 光刻机

- 第52节 中国的人造卫星时刻？：中国政府正在努力减少对美国的依赖，大力支持长江存储之类的芯片公司

- 第53节 短缺和供应链：由于芯片需求不断增长，各个科技大国都在增强半导体供应链能力和芯片制造能力

- 第54节 台湾困境：一旦台湾出现危机，将给半导体行业造成巨大损失
  - 美国所依赖的芯片制造能力大多在解放军导弹的射程之内

## 读这本书之前我知道什么

读这本书之前，我已经读了《了不起的芯片》，因此对芯片行业已经多了一些认识，包括：

- 芯片生产流程最重要的两个环节是芯片设计和芯片制造
  - 芯片设计的流程包括前端设计、芯片验证、后端设计等
  - 芯片制造的流程包括制造晶圆、光刻、腐蚀等
- 芯片公司分为三类：只做芯片设计、只做芯片制造、两者都做
  - 只做芯片设计的公司有：英伟达、华为海思
  - 只做芯片制造的公司有：台积电
  - 两者都做的公司有：英特尔
- 目前只有阿斯麦公司（ASML）能生产 EUV 光刻机（EUV 是极紫外光的意思）
